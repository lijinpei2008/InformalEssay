# 第十三章 异身同体的梦想——设置台湾自治议会请愿运动

## 引言部分
- 1920年前后，台湾开展了为了反抗日本总督的压迫和获取政治权而进行的政治运动——台湾自治议会设置请愿运动，该运动被称为`过渡时期诞生的稳健型抗日运动`。
- 在1920年，日治初期，台湾以传统价值观为根基的武力抗日斗争陷入绝路，而共产主义与工人、农民运动即将兴起。
- 起始于1914年，在1920年中期到达高峰，1934年结束，该运动是当时的台湾人`唯一现代化且合法的政治活动`，目的是向日本帝国议会请愿，希望能够在台湾设立自治议会。
- 是一种以最初始的形式探索有别于和`既存的民族国家之间进行统合`或是`建立新的民族国家`之间的第三条路。

### 如何理解“过渡时期诞生的稳健型抗日运动”？
```
个人看法：
过渡时期指的是1920年前后，台湾以传统价值观为根基的武力抗日斗争陷入绝境，共产主义和工农运动方兴未艾。
稳健型指的是以和平请愿，非暴力的方式将自身的政治诉求传递出去。
抗日指的是这项请愿活动的目的是为了打破日本总督府在台湾实行的独裁统治，使得台湾人获得作为“日本人”的权利。
```

## 为获得权力的“同化”
### 台湾议会设置请愿运动的前身——台湾同化会
台湾同化会的一生：
- 1907年，梁启超和林献堂相识于日本，梁启超告诫林献堂“中国今后30年间没有救援台湾的余力，因此必须效法爱尔兰，在武力斗争败北的同时，改为向英国舆论呼吁陈情，以合法运动的方式来获得参政权。”
- 林献堂之后又从某位国民党干部了解到“台湾住民为了提升地位而向日本陈情，以获得理解和支持，这种文件的运动最为恰当。”
- 1913年春天，林献堂在东京拜访时任内务大臣原敬，失败。
- 在拜访原敬失败后，改为拜访以自由民权运动为旗手的板垣退助。板垣退助对台湾人民的遭遇深表同情。
- 1914年2月，板垣退助视察台湾之后，提议设立以“本国和台湾融合”为宗旨的台湾同化会。大隈重信和东乡平八郎等人表示支持。
- 1914年11月，林献堂拜访内田民政长官，长官叮嘱“同化会立意虽好，但政治就像利刃，如果任由小孩随意玩弄，那是极度危险的”。
- 1914年12月，板垣退助再次造访台湾，并以会长的身份在台北成立了以“内台融合”为宗旨的台湾同化会，成立之后集结了三千余名参加者。首相大隈和总督府民政长官内田嘉吉送来贺词，总督府诸多官员也出席了仪式。
- 1914年12月底，板垣退助离开台湾后，总督府以“经运不正”为由逮捕了理事。
- 1915年1月，总督府以“有害公安”之故，强行解散了同化会。

林献堂在成立同化会之初，四处游说日本中央人士，其观点内容大致为：“自日本领有台湾以来，台湾总督便凭借着‘六三法’掌握独裁权，使得台湾变成‘宛如和日本帝国毫无关系，而是总督府的台湾’。尽管天皇高唱‘一视同仁’，但总督府依旧不断镇压台湾人民，和邻近由美国统治的菲律宾相比，也仅有基础教育设施。若再放任这种状态下去，‘不管是为了台湾，还是为了日本国家，最后都不是良好之策’。”
```
在这里，林献堂把帝国议会希望收回独裁权的总督府当成了敌人，表示了身为台湾人民的代表希望和帝国同进共退拥护天皇的忠贞思想，获得了日本中央人士的支持。
```
对林献堂到访表示关心的政治家包括了尾崎行雄、岛田三郎、犬养毅、大隈重信等。和接受谢花升陈情的政治家是同一批人，都是对六三法不满的政治家，但对于林献堂和台湾人民的境遇也仅仅是口头上表示同情。

### 同化会会长板垣退助、以林献堂为代表的台湾人、台湾总督府各自的立场：
板垣退助的思想是典型的大亚洲主义（泛亚主义），将自己和大日本帝国描述成反种族歧视和反帝国主义的斗士，因为总督府的压迫导致了日台融合进展缓慢，希望安定台湾统治获得台湾民心，从而将台湾拉入日本阵营，从而使之成为日中亲善的桥梁，共同抵抗欧美白人，根本目的是为了获得对天皇忠心的臣民，毫不关心台湾人的政治权利。其政治思想为：
- 强化台湾人国语和修身教育
- 迁移至少百万内地人去台湾定居通婚·
- 缓和总督府压迫政策，使内地人和台湾人相融合
- 日本领有台湾之目的是为了保障国家安全。

以林献堂为代表的台湾人在孤立无援的情况下，通过重新解读“一视同仁”这一个国家基本精神，获得了政治正确的诉求，即日本方面无法直接的否定台湾人民的同化请求。但是，因为其政治诉求的根本目标是获得台湾人自己的独立议会，所以这种与总督府、帝国中央议会相冲突的政治诉求从根本上而言是不可能实现的，看似如火如荼的台湾同化会为什么只能生存一个月，为什么持续了13年的台湾请愿运动最终失败之根源就在于此。

台湾总督府面对台湾人民提出的饱含政治正确的口号“日台融合”，无法直截了当地提出反对，如果胆敢明面上提出反对意见，必然会被帝国中央议会找到把柄和借口，从而顺理成章的收回总督府的独裁权、立法权，废除六三法等。所以在面对拥有国内颇具影响力的政治家支持却依旧处于绝对弱势地位的台湾人所成立的台湾同化会，总督府只需等待国内政治家回国后，便可以随心所欲地以符合规则的方式强行解散台湾同化会，从而保全自身的政治利益。


### 如何理解“台湾人居然会为了和总督府斗争，高举天皇‘一视同仁’旗帜，还向大日本帝国的中央政治家陈情，这种手法乍看之下相当不可思议”
```
之所以乍看之下觉得“相当不可思议”，是因为从整体而言，日本和台湾是对立的状态，即对台湾人民而言，日本总督、日本议会、日本政治家都是敌人。
但是正如梁启超所言“中国今后30年间没有救援台湾的余力”，台湾在二十世纪前中期，处于孤立无援四面楚歌的情况下，面对日本总督府的压迫，面对武力斗争的失败，只有在日本国内寻求和总督府敌对的政治集团的庇佑，才有能力去反抗总督府的压迫从而改善台湾人民水深火热的生活。
毕竟敌人的敌人就是朋友。
```

### 如何理解“这一运动理所当然在某种意义上在‘抗日’的同时，又带有‘亲日’的微妙性质”
```
抗日：台湾的总督府这个政府机构是日本设立的，因为反抗总督府的压迫，所以说是一种“抗日”运动。
亲日：孤立无援的台湾为了反抗总督府，只能以加快日台融合的方式，在日本国内寻求帮助，所以说是“亲日”运动。
```

## 对多样性的期盼
台湾传统教育到现代化教育的转换：
- 1898年-1918年，台湾传统教育机构书院的数量锐减20%
- 1912年-1920年，台湾公立学校就学率从6.6%提升至25.1%

台湾文化发生的转变：
- 1900年，成立天然足会
- 1911年，成立断发不改装会

台湾与琉球：
- 1900年，冲绳青年之间流行和服
- 1920年，台湾青年之间流行西装和上海服

台湾作为“文明化”的一环，应该要对老旧的风俗进行改良，对于上述转变，日本知识分子和总督府都表示了支持。但由于中国也有断发放足等运动的兴起，所以断发放足并不是有力的证据能够证明台湾“日本化”。对于日本而言，台湾青年脱掉“支那服”换上西装仅仅是“文明化”但并不等于“日本化”，至少是改穿`和服`这个阶段才行。

1915年，西来庵武装抗日运动，以日本方面大量虐杀和处死九百人的结果告终，标志着台湾以`传统文化为根基`的`武装反抗运动`走向灭亡，开始转变为以`现代化为根基`的`合法且现代`的`和平请愿运动`反抗形式，其先驱是`台湾同化会`以及`中学设置运动`。

作为被称之为台湾最现代化的阶层——内地留学生们，对中学设置运动最为敏感。因为对高等教育的追求，内地留学生群体在 1915年-1919年-1922年期间，人数从300余人增加到800余人再到超过了2400人。

在英法等国的殖民地，从宗主国留学归来的人物多半成为现代抵抗运动的核心。在日本，则出现了：
- 冲绳推动参政权获得运动的谢花升就是从内地留学归来的
- 朝鲜三一运动的独立宣言，是由朝鲜的内地留学生独自起草的
- 台湾议会设置运动，是由台湾的内地留学生推动的

在`1919年之前`，台湾留学生群体虽然都“很努力和内地风俗习惯进行同化，对于社会问题、政治问题并没有表现出深刻的关心”，即便是被人嘲笑也没有人挺身而出进行反驳，尽管大家对台湾现状感到不满却无法提出一个有效的运动仿真。

`1919年之后`，受第一次世界大战结束后民族自决的风潮，朝鲜三一运动以及中国动向的影响，又因为台湾不具备原本是独立国家的历史脉络，再加上流血镇压的过往记忆，所以台湾留学生群体没有像朝鲜人那样高唱“马上独立”，而是持续着合法与非暴力的路线。

### 如何评价台湾留学生的行为？
```
在转变前，台湾留学生群体的不作为与不敢作为是因为其生活条件的优渥导致的，无论外界如何变化，只要能够保证其优渥的生存环境和学习环境，生活在象牙塔中的学生是不会在意外界一切事物的。
大部分留学生学成返乡，开始发现他们虽然还是处于受人尊敬的位置，但是丧失了参政议政的权利。从受人尊敬的社会精英跌落人间成为了丧失主权寄人篱下的平民，这种巨大的转变加上外界民族自决民族独立运动的兴起，诱发了这种转变的发生，希望通过一种社会运动进行社会变革。
我们都知道，革命不是请客吃饭，没有不流血的革命。革命就是用暴力的方式从已有的利益集团手中抢夺和再分配利益。自认为天之骄子的台湾留学生群体，自认为高人一等，不愿意和社会底层的泥腿子一起流血牺牲，异常惜命的他们希望通过用合法与非暴力的请愿运动去祈求既得利益者施舍一部分利益，对于既得利益者而言这无异于钝刀割肉，所以是绝不可能成功的。
身为小资产阶级的台湾留学生群体其阶级的软弱性与妥协性，在这个运动里体现的淋漓尽致。
```

新民会和《台湾青年》杂志：
- `新民会建立的起因`：1918年，东京中华料理店，林献堂和留学生们一致认同“不管同化还是非同化，六三问题始终都束缚着台湾人，应该要将它撤销才对。”
- `新民会和《青年杂志》的诞生`：
    - 1920年1月，台湾留学生组成了“新民会”
    - 1920年7月，“新民会”发行了机关刊物《台湾青年》
- `新民会纲领`：仅提倡“文化向上的提升”，但在干部会议上决定开展“台湾统治的改革运动。”
- `新民会主张`：废除六三法
- `《台湾青年》杂志停刊`：因后来改名为《台湾新民报》，自动停刊
- `《台湾青年》杂志特征`：
    - 日本方面投稿者占据了不小的比例
    - 超过一半以上的内容以日文呈现
    - 台湾人的投稿也是日文居多
    - 以对话路线为主轴
- `《台湾青年》杂志的立场`：既要推进“文明化”，又要拒绝“日本化”

### 如何理解《台湾青年》杂志内容以日文居多的现象？
```
外部客观因素：该杂志受到内务省与台湾总督府的双重检阅，创办“新民会”的留学生群体是接受了日语现代教育的一群人。
内部主观因素：因为《台湾青年》杂志的发行者和投稿者是留学生、通过日语教育吸收“文明”的一群人、日本国内学者，所以在这些人的潜意识里，将“日本化”和“文明化”作了比较，虽然“文明化”不一定是“日本化”，但是“日本化”却是“文明化”的一部分。所以是用日文发表文章是文明的象征，是知识分子的象征。
后果：因为一多半的内容是用日语表达的，所以这本杂志在台湾的普及就受到了很大的限制，对于绝大多数人而言，这本杂志的内容是看不懂的。因为看不懂杂志所表达的思想，所以留学生群体的思想对于台湾人民来说是接受不了的，因为他们不懂。也就是说在留学生群体的家乡——台湾，绝大多数民众是不会支持他们的思想和行动的，我们也可以从请愿运动参与人数最多也没有超过三千人这份数据中分析出来，他们已经严重的脱离了人民群众，他们所倡议的一切行为都是空中楼阁，必将遭受失败。
大半内容用日语编写的以对话路线为主轴的《台湾青年》杂志是谁和谁对话？是留学生和日本的对话，或许在留学生眼里大部分台湾人民并不是“文明人”，不具备“文明化”的人是没有资格参与对话的。
```

### 日本人投稿强调的内容
以对话路线为主轴的《台湾青年》杂志上投稿的日本作者，类型相当广泛，反映了三一运动的氛围，几乎一致认为朝鲜和台湾的统治机制到了必须要改革的时候了。
- 植村正久和海老名弹正等基督教知识分子
    - 海老名弹正批判：“台湾教育设施不够完善，对于欲跻身世界文明国之列的日本来说是极度不像话的。”
- 永井柳太郎和岛田三郎等民权派政治家
    - 永井柳太郎主张：“日本和台湾的关系，和英国对印度的关系不同，是属于同一文化系统的同一人种。”
    - 岛田三郎表示：“日本天皇的圣意是对所有人平等的施予善政，所以日本面对西洋诸国提倡的普世人种平等的观念，应该使用于台湾。”
- 吉野作造和末弘严太郎等大正民主派
    - 因提倡“一视同仁”和“文明”的普及来化解歧视论，与同化主义的批判同时并立，从而呈现出一种`文化多元主义`的主张
- 安部矶雄和佐野学等社会主义者
- 三宅雪领与安罔正笃等亚洲主义着
- 台湾总督田健治郎以及总督府总务长官下村宏
- 早稻田大学教授内崎作三郎
    - 内崎作三郎表示：“日本民族是混合民族，应当促进日台融合。”

日本作者论述的内容，主要集中在
- “一视同仁”式的反歧视批判
- 反对六三法问题以来的总督府特权
- 高唱对抗欧美的大亚洲主义
- 主张把台湾当作“日中亲善”的媒介来重视
- 批判内地殖民者的横暴与歧视态度，以及外出捞一票就走的过客心态

### 台湾人投稿强调的内容
从结论来看，台湾青年当初的论调表面上和日本方面并没有太大的差异，主要强调的是第一次世界大战后“民主”与普遍的“文明”，并对总督统治的专制性质加以批判。与日本作者的不同之处在于大多数人认为“台湾文化远远无法和世界文化相比，至少落后了2、3个世纪，保持着台湾恐怕会成为文明的落伍者的危机感。”

高举“去除陋习，吸收新文明”的旗帜，内容包括：
- 家族制度的腐败
- 卫生思想的地下
- 鸦片吸食的陋习
- 女性地位的低下
    - 《台湾青年》杂志几乎每一期都有关于提升台湾女性地位的投稿被刊登出来
    - 反对：畜妻制、买卖婚姻、缠足等制度习俗
    - 提倡：恋爱自由、婚姻自由、女子教育普及

汉民族认同的主张则是并不把日本看作“文明”的体现者。他们和那些企图给家乡带来“文明化”却又对宗主国保持着反抗心理的人有着相同的特征，他们认为：
- 日本的一切都是模仿的
    - 文化、政治、法律、文学、美术、音乐
- 保持着否定“日本化”的态度
    - 女性地位的提升比起国语（日语）的普及更为重要
    - 和服与生鱼片都不适合台湾的亚热带气候，在台湾推广和服与生鱼片是对重视卫生与机能的“文明化”大开倒车

台湾留学生群体几乎都是台湾原住民当中的上层阶级，所以他们提倡推进“文明化”，又拒绝“日本化”。
- 虽然批判同化政策，但对总督府推行的经济开发和卫生整顿政策表示支持
- 一边批判教育片中日本语，同时又感叹日语识字率低落
- 认为和内地人共同学习是废除歧视的一环加以提倡
- 高呼提升女性地位废除蓄妻制的同时，反对日方决定将本国的民法、商法、以及破环传统家族制度的亲族法延长到台湾

因为留学生对于欧美的态度也是处于爱恨交织的矛盾状态——他们认知到欧美是“文明”的先进国家，并倾心于罗马拼音和世界语；但在另一方面，他们也很难忘记欧美是分割中国、歧视有色人种的侵略者——所以用“同文同种的日华”这种结论的大亚细亚主义也能在《台湾青年》杂志中寻找到踪迹。

也有住在美国加州的台湾人的投稿，一边传达受遭受排挤运动的日裔移民苦难的境遇，一边却又在对日本保持敌意的同时表现出对欧美的亲近感。即便是主张日中连带说，也是以关系对等为前提。

也有一部分留学生群体以身为汉民族而自豪，面对亡国之虑悲愤交加，心忧社会民族，提出“敦促民众奋起、自助、努力”。

对于自己是否是“日本人”，关于这点，这帮留学生们的心态同样的复杂。一方面提倡汉民族认同，一方面又把自己定位在“日本人”的位置上。这种对于自身集体的自负与自卑感，还有“文明化”的推进论，在创造冲绳民族主义的伊波等人身上也是可见的论调，对被自己视为更低贱者的歧视，也是如影随形的存在着。台湾留学生一边谴责日本方面的歧视，一边却又主张汉民族认同和无历史、未开化的生藩截然不同。露骨地体现出对原住民的歧视，这样的人也是存在的。

### 蔡培火投稿强调的内容
蔡培火初到东京对日本满怀“民族的憎恶之情”，后受日本基督教牧师植村正久的感化放下了这种感情，加入基督教以“四海兄弟主义”为信条，认为台湾是日本的一部分。

蔡培火主张在达成制度平等的同时，也要保障善良的文化。为了将“文明化”和“日本化”分离开来，他在《台湾青年》杂志发表一篇名为《吾人的同化观》的论著。他认为同化分为`自然同化`和`人为同化`，人为同化方面的内容无论怎样努力都不会成功。成功的唯一方法就是保持着“异身同体的精神”，去“尊重个性、保障善良的文化”，并抛弃过去的歧视观念和领土的野心。在他看来，现实的台湾统治却是充满了歧视的，总督府所谓的同化不过是“榨取主义的别名”、“愚民政策的表面招牌”罢了。
- 自然同化
    - 废除缠足、禁吸鸦片、等`本质性的陋习`，以及促进`人格尊重自由平等`，这些以`全世界规模开展`的事物
- 人为同化
    - 奖励和服、街名改为内地风格、门口竖立门牌

和单纯的抱着“民族自负心”的人比起来，蔡培火反而能够从那种对抗强者的民族主义在面对弱者时所产生出的歧视链种逃脱出来。

### 文明化与日本化的关系
```
文明化指的是：学习先进的文化、知识，学习科学的卫生习惯，提升女性地位，普及教育，提倡自由、平等、公正、法制等。
日本化指的是：有选择地学习先进的文化、知识、卫生习惯、衣着、普及教育等，着重学习日本传统文化、日本语、饮食习惯、衣着审美等一系列日本风味浓厚的相关内容，从而实现文化上的融合。
日本方面在面对殖民地的时候将“日本化”和“文明化”划等号，希望将当地原住民从身穿本地传统服饰转变为西装领带，再转变为和服。通过这种形式的转变，让日本是祖国的观念、忠于天皇的观念深入当地人心，实现民族的同化和融合，方便后续的统治。
台湾留学生群体几乎都是台湾原住民当中的上层阶级，他们反对将“文明化”和“日本化”划等号，希望通过“文明化”来获取知识和力量，又反对“日本化”来追求和保全自身的权利。没有人会希望自己的头上有一个掌握着自己生死大权的统治者。
```

## 对殖民政策学的置换概念
置换概念即偷换概念，是在思维和论辩过程中自觉或不自觉地违反同一律的逻辑要求，用一个概念去代换另一个不同的概念而产生的逻辑错误。偷换概念也是一种常见的诡辩手法。主要有以下几种表现：
- 任意改变一个概念的内涵和外延，使之变成另一个概念
- 利用多义词可以表达几个不同概念的特点，故意把不同的概念混淆起来
- 抓住概念之间的某些相似之处，抹杀不同概念的本质区别
- 混淆集合概念与非集合概念

林呈禄一边肯定政府和总督府迄今为止维护六三法的根据“基于特殊民情风俗的特别立法……比起用作维持律令权的根据，这反而更应当设置特别代议机构的理由”，将它置换成了自己主张的立论基础，在此同时殖民政策学里的同化主义批判，也被利用来当作独立性的立论根据。

当时并不存在“多元主义”生存的土壤，他们也仅仅是要求作为“日本人”的权利，表现出来的方式必然是以和日本方面最低限度的摩擦为界限——通过置换概念，用支配者的言论概念去争取自身的利益。在这里，台湾留学生们选择的支配者论述是——殖民政策学。
- `起因`：留学生选择的方针是“撤废六三法”，这是继承自台湾同化会路线
- `经过`：
    - 1920年11月，蔡培火等人在以此演讲上将“撤废六三法”和“给我们自治权”两件毫不相干的事情随意的结合起来
    - 1920年11月，田健治郎总督前往东京，表达了想要维持总督立法权的意图
    - 1921年1月，永久法“法三号”立法通过，林献堂和留学生们主张的“撤废六三法”方针遭到了背叛，运动完全遭受到了挫折
    - 明治大学法学科学者泉哲是一名基督教徒，曾在《台湾青年》投稿表示“文明殖民国若要保持其地位，必须以考虑殖民地人的福利为统治的着眼点”
    - 林呈禄在《台湾青年》投稿初期，主张以“日本人”的身份去寻求撤废法律的歧视。
    - 受到殖民政策学的影响，林呈禄认为英国成功统治加拿大和澳大利亚的“自治主义”是殖民政策的国际潮流，开始主张台湾自治问题。在《台湾青年》发表了一篇名为《六三问题的归著点》的文章，提出“台湾应该获得自治”
- `结果`：新民会内部对于“六三法撤废”改为“追求自治主义”方针的讨论，最终由林呈禄等人引用山本美越乃的著作为论述基础，力陈“自治主义”的好处，终于取得了胜利
- `自治主义的实质`：在承认日本主权的情况下，总督只是单单昭示于一般统治的大纲，至于其实质及内容，乃是由殖民地住民自己决定，只有在面对第三方国家进行外交的时候，由母国日本出面主导
- `请愿运动的初始`：蔡培火和林献堂赞成放弃六三法废除运动，开始主张“自治主义”。作为“自治主义”的第一步，决定向帝国议会请愿，请求设立由台湾住民选举产生的台湾议会
- `美好的误会`：
    - 当年日本引进欧美的殖民政治学时，已经因为学者的不理解和欧美形式的错综复杂，产生了相当大的误会
    - 台湾留学生又把自己的希望寄托于这种错误的理论学术上，囫囵吞枣般的理解和应用，于是这种误会就更加深重了
    - 思想是为了人而存在的，只要能够完成人们心中现在的愿望，对思想的故意曲解是必然会发生的事情，台湾留学生虽说学习到了错误的思想，但是却歪打正着的创建了解决自身愿望的指导思想——“自治主义”

### 如何理解“台湾留学生会去学习作为支配着学问的殖民政策学，这本身就是一个相当奇特的现象”
```
原本是希望通过换位思考的方式，从“日本人”的角度提出一项切实可行的、减少对台湾人民压迫、提升台湾民众利益的政策方针。从这个角度来看，毫无疑问，学习正确的殖民政策学可以实现这个目标。
但是没想到的是，日本的殖民政策学本身的理论基础就是错误的，而留学生的学习又是囫囵吞枣，知其然不知其所以然，错上加错的情况下诞生了一种不切实际的想法“自治主义”
```
>1867年，英国议会通过《1867年英属北美法》(British North America Act, 1867)，将北美的三个移民殖民地加拿大省（Province of Canada，后来分为安大略和魁北克两个省）、新不伦瑞克（New Brunswick）和新斯科舍（Nova Scotia）合并组成加拿大自治领（Dominion of Canada），并规定，加拿大自治领选举产生自己的议会（Parliament of Canada），在自己的内部管理事物上行使立法权，英国议会和政府保留宪法性事务、外交和军事方面的权力；同时规定，加拿大模仿英国的政治制度成立自己的政府。政府不再由总督任命，而是依靠加拿大议会的支持而存在，议会多数党的领袖担任加拿大总理，管理加拿大的内政，总督则代表英国政府管理加拿大的外交和防务。

### 学者泉哲
明治大学法科学者，原本是专攻国际法的专家，性格相当温厚的基督教徒。曾在《台湾青年》投稿表示“文明殖民国若要保持其地位，必须以考虑殖民地人的福利为统治的着眼点”，还提出“使用暴政压迫或掠夺等‘本国本位’政策的国家‘没有资格列入文明国家之林’”。

虽然泉哲这种论调是欧洲支配殖民地的意识形态，但是对于努力挤入“文明国”行列的日本而言，就变成了一种具备统治批判论的机能，指责其未尽到“文明过”的责任的论述了。

泉哲自己本身对于这种论调也是打心底相信的。其中一个原因是：
- 身为国际法学者的他，认为第一次世界大战之后和“国际联盟委任统治地”相关的联盟规章既然这样设定的，那么加盟国就理应遵守这样的国际法规才对

泉哲和其他学者一样，不形容台湾人为“日本人”；在称呼台湾为“殖民地”的同时，也以英法等国放弃同化主义的殖民政策这点为根据，对同化政策进行批判。

完全没有受到人种思想导致的同化主义批判论的影响，而且他对法国的同化主义批判逻辑一无所知。

另一方面，他把日本归纳为“自治主义”之列，同时也把加拿大和澳大利亚等自治领，和突尼西亚之类的保护国全都归类到了“非同化主义”之中。和吉野作造一样，在批判同化主义的同时，也主张撤废内地人与台湾人的差别待遇。
```
和山本或矢内原等其他殖民学者相比，泉哲是一位相当程度反映了大正民主暧昧性格的人物。他的殖民政策学当然对统治台湾和朝鲜的学理完全排不上用场，甚至也不能说对理解欧洲殖民地支配事态有所助益，但是他的问候性格与殷切互动，都让台湾留学生深受感动。
```

### 如何理解自治主义的实质
```
对外承认台湾是日本不可分割的一部分，在对外的国际外交中，由日本出面进行交涉。
对内则是台湾独立自主，总督只是吉祥物一般的人物，台湾内部一切事物都是由台湾人自己进行管理和统治，或许会定期向日本进行纳税？若是出现财政赤字也有可能会向日本寻求财务援助？
从某种程度上来说，这种自治主义和封建时代上贡称臣的藩属国没什么区别，甚至在当时的时代背景下有一种把日本当作召之即来挥之即去的冤大头靠山的感觉。这种想法过于天真。
```

### 如何看待台湾留学生通过置换概念来争取自身的政治利益
```
失败是必然的。
原本觉得“废除六三法”能够缓解日本人对台湾人的压迫，随后却发现如果台湾独立自治带来的利益会更加的庞大的留学生们在对社会毫无威胁和影响力的情况下，没有考虑实际的情况下于二十世纪二十年代提出了这种
把日本当作冤大头的理论路线，并开始拥护之前反对的六三法，并通过偷换概念的方式拥护台湾人自己的“六三法”，希望通过请愿的方式将这套扭曲的理论灌输到日本统治者的脑海中，从而实现“台湾的独立”。
台湾新民会本身不具备任何武装反抗力量的同时，又没有获得广大人民的支持。像这样没有武装力量和舆论影响力的政治党派或集团，对日本议会，日本总督而言是不会放在眼里的。
面对新民会最多也只有不到3千人的请愿运动，对于日本方面而言这是新民会的请愿运动，不是台湾人的请愿运动。在毫无外界压力的情况下，这种糟心的请愿运动和请愿书，束之高阁拖着就够了。
```

## 基督教徒与大亚洲主义者
设置台湾议会这一面向帝国议会的请愿运动，必须要有日本方面的医院引荐才行得通。
为台湾新民会请愿运动提供帮助的议员有田川大吉郎和清濑一郎，以及之后的神田正雄。
对于田川和神田而言，他们的思想不过是为了正当化大日本帝国支配而高唱的“说教”，但和其他的政治家不同的是，他们打心底相信和付出了实际行动去实践它。

### 基督教徒田川大吉郎
- 以中国语翻译身份参加了中日、日俄战争的新闻记者
- 乃木总督时代，出任台湾报社主席
- 虔诚的基督教徒，植村正久的教徒，和蔡培火有一份同窗情
- 当选众议院议员后，致力于解决都市问题

他的论调是那种“文明传播论来正当化支配”的老调，且用在了批判统治上。他认为“现在实行的所谓同化，与其说是‘文明化’不如说是‘日本化’更恰当”，并提出“不仅使用日语也要用外国语进行世界知识的教学”。

希望对台湾的统治借鉴英国对印度的自治统治策略。他认为最先进的国家，实行的理应是文明且人道的统治。认为日本是文明的“后进国”主张内地和台湾都全面废除汉字，改用罗马拼音。他赞同台湾议会设置请愿，主要的逻辑也是出于“将象征文明的恩惠的议会政治赋予台湾，是日本的使命”。

赞同“英国式”的统治认为英国是`文明的体现者`的世界观，反对将日本宪法延长到台湾的同时又采纳了“同一的日本人”与自治论两个论述。

## 多元的日本、多元的台湾
### 第一次请愿运动始末
第一次请愿运动清楚地展现了这个运动的基本性格
- `开端`：大日本帝国是立宪法制国家，台湾是归属于帝国的版图的一部分
- `认同`：台湾人是日本人，也就意味着全体“日本人”应改要依据宪法受到人权保护才对
- `讴歌`：台湾议会的开设是完成其地理上、历史上的特殊使命
- `目的`：追求作为“日本人”要求权力的同时，希望保持不同于“日本人”的独特性

面对这次请愿运动，总督府在东京辨事处传唤了林献堂、蔡培火、林呈禄等人，总督田健治郎亲自说服他们停止请愿，告知他们《法三号》的备案情况，同时设置总督府评议会的天也会复活并且实施，并将启用台湾人担任评议员。双方因为理念差异没有谈拢。
- 这个评议会和六三法时代与原敬妥协下的状况有所不同，它被排除了对律令的议决权，只是单纯的咨询机构职能，而且评议员也是总督任命制
- 而请愿一方想要的台湾议会是：“拥有关于台湾立法及预算的议决权，且以公开选举产生的议员为骨干组成”

田健治郎觉得自己礼贤下士却被人驳斥面子受损，在商议法三号时断言“台湾自己设置立法议会是绝不可能的。”政府方面也暗中照会两院的请愿委员会，希望他们不要接受这个请愿，于是不管是贵族院还是众议院，在商议都未曾进行的情况下，直截了当的拒绝了。

随后，总督府立即发表了反驳请愿运动的谈话:
>台湾岛民乃是“同文同种”，日本统治台湾并非是如欧美统治殖民地“只是为了获得经济上的资源”，而是抱着“一视同仁”的宗旨，朝着使台湾称为“纯粹的帝国领土的延申”而努力；故此，以英国的“自治主义”为先例，要求设立台湾议会，这与“我帝国统治台湾的这一根本方针不相容”。

### 失败之后
虽然第一次请愿运动失败了，但是运动者的一直和凝聚力却越发坚固。当时的内地知识分子和言论界普遍对请愿运动的保持着好感。加之朝鲜独立运动之后产生的危机意识，大日本平和协会等机构明确主张“既是认同日本的主权，那么不管是怎样的内部大改革方案，我们都应该不遗余力的加以协助。”就请愿者而言，此时他们唯一的敌人只有台湾总督府，而内地则是自己的友军。

但是对于从内地移民到台湾的日本人而言，反对呼声越发强烈，内地人经营的报纸开始统一口径齐声批判请愿运动，林献堂家中也受到了恐吓信。

他们认同自己是“日本人”，主张请愿是“国民的要求”，除了是应付总督府的镇压之外，又有多少真心话呢？与此同时，还强调“台湾议会是由不分住在台湾的内地人或本岛人，或是行政区域内的熟藩人，平等公开选举出来的代表所组成”。呼吁内地移民者至少要将“台湾当成子子孙孙永久居住的地方，为促进台湾发展尽一份国民的责任。如果有这种自觉的化，必然会不甘居于制度之下而是应当与岛民结合，要求立宪政治”。

## 违宪的界限
1922年2月，汇集了连署书的第二次请愿，请愿委员会依旧是在没有讨论的情况下将其驳回
- 1921年春，回到台湾的林献堂、蔡培火等人以刚刚创建的台湾文化协会为运动基地，在台湾四处奔波，为了普及科学知识与“新文明”而开展了巡回演讲。自1923年到1926年，文化协会共举办了798场演讲，听众累计30万人
- 1922年9月，林献堂拜访了总督田健治郎，田表示“请愿是宪法赋予我臣民的权利，我不会干涉请愿运动，但是出于朋友的角度，劝你还是停止请愿吧。”
- 1923年以后，总督府更是将内地的`治安警察法`延申施行到台湾，据此对集会和出版加以限制
- 1923年2月，蔡培火等人成立的“台湾议会X同盟会”就遭受镇压
    - 申请公认结社认可被拒的情况下，同盟本部转战到日本东京
    - 同年12月，总督府将居住在台湾的同盟成员一并逮捕

1923年2月，第三次请愿
- 来到东京的蔡培火一行，受到挥舞着“自由”、“平等”、“台湾议会”等小旗的台湾留学生热烈欢迎
- 他们一再强调运动的合法性、立宪政治的台湾延长，以及作为“国民”的权利
    - 通过西餐厅招待新闻记者、引荐议员
    - 在市区沿途散发五颜六色的宣传单
    - 让台湾飞行员从空中分发传单
    - 将请愿理由书这本小册子分发给两院议员及报社
- 其结果就是总督府和政府对此运动大表敌视，帝国议会对此运动少有赞成意见
    - 1923年12月，面对总督府逮捕台湾同盟成员一事，议员们抗议总督府侵害人权，但也未必就会赞成请愿主旨

请愿运动是以和民权派议员一直敌对的“总督府的自治”同样的论述方式发展出来的产物，而在台湾设置自治议会，无异于成立“第二个帝国议会”，必然会是的帝国议会的权限遭到限制。

### 请愿运动的弱点
```
请愿者试图逆转原本属于支配者的殖民政策学，利用总督府用来使特权正当化的“台湾特殊性”逻辑，展开了这个请愿运动。
但是日本内部的支配着是分裂的，他们情愿的对象——帝国议会，采用的是和总督府不同的方案——内地延长主义。
不止如此，随着永久法《法三号》的实施，保全了自身特权的台湾总督府也改换门庭，开始接受内地延长主义，于是站在请愿运动阵营一边的，就只剩下一部分的殖民政策学者以及言论界的部分人士。
```

### 如何理解这种敌视
```
总督府的特权原本就是日本议会要打击的目标，在1921年4月永久法《法三号》正式实行之后，对日本议会而言这种明显违宪的法案，是文明史上的污点，不单单是绝不可能再次出现，还要捂严实了不能让欧美看笑话。
请愿运动却用“既定的特权”去谋求属于自己的“自治特权”，还在行动上如此的大张旗鼓，这对于帝国议会而言无异于当众揭老底的打脸行为，对于总督府而言这种行为有可能会让自己利益受损。
因此才会遭受双方敌视。
```

1924年1月，第四次请愿，由于碰上了议会解散，未能上呈。

1924年7月，第五回请愿，因田川大吉郎落选，引荐议员由神田正雄和清濑一郎担任。
- 原本请愿目标是朝着完全自治的方向进行，但是请愿书上却只写道“总督实行的立法、以及台湾相关的预算”由协商权
- 请愿者中也有一部分人提出了“台湾议会就是相当于内地的地方议会”这种说法。
    - 这是一种当帝国议会将台湾议会解释为侵权，请愿绝不会通过的情况下，无奈之中使出的战术
    - 但是在计划书中，对于议会的具体构成与选举方式、权限范围等完全都没有明确的描述
    - 简单来说，台湾议会这个词，是将台湾人模糊隐约的愿望，与内地方形形色色的统治改革论绑在一起的象征，却又不存在十分具体的内容

从殖民政策学的暧昧里产生出的“自治主义”在运动初期能够起到集结众多人的支持，发挥了很大的作用。可是一旦请愿碰到了阻碍，那么这个自治外强中干的真相便一下全都暴露出来了。

1925年，第六次请愿，请愿人数达到了迄今为止最高的782人，并重新附上了一篇洋洋洒洒的“请愿理由大纲”
- 大纲中依旧没有明确说出台湾议会的具体构成蓝图，只是一味的批判台湾总督府的压迫性政策，并强调请愿是“符合立宪思想、有合理一句的国民要求”而已
- 台湾总督府警务局针对这次请愿，拟定了一份反驳的内部文件
    - 它对请愿运动的理论基础——“自治主义”——提出批判，并看穿了它的虚构性
    - 希望对请愿运动进行镇压之外，试图说服运动领袖，诱导他们遵循内地延长主义，走上“参政权获得运动及地方自治制度的改善运动”这条道路
    - 要赋予实际的参政权，当然也要在“议员自不用说，就连选举的住民都充满着“忠实奉公牺牲”精神的大前提下，采取渐进式在长远将来由总督实行立法权通过才行

1926年，第七次请愿，请愿人数达到了两千人
- 时任首相若槻礼次郎说特别议会的设置“违反了宪法”，“请愿者对于他们要求的是殖民地自治还是和内地一样设置地方议会根本模糊不清”随后单方面驳斥了请求

## 分裂的请愿运动
1927年，因为多次请愿失败且内部对立日益激烈，台湾文化协会正式分裂
- 受共产主义与无政府主义影响的左派，后来掌握了协会的主导权
- 蔡培火等干部脱离协会组成台湾民众党延续请愿运动
- 林献堂
    - 1922年，因受到田总督恐吓便推出了运动
    - 1927年，在协会分裂后，前往欧美各地漫游
    - 之后以养病为由定居东京，两年不会台湾

运动分裂造成了重大的打击。掌握文化协会主导权的左派，形容台湾议会请愿运动是`一味的恳请资产阶级代仪士施予小恩小惠、汲汲营营与和日本主义妥协`，并说“我们唯一能做的要么是打倒那些家伙，要么是被那些家伙打倒”摆出了明确的对决斗争方针。
- 公然高喊口号要求内地人和台湾人“打破共学”
- 山川均等内地左派势力也开始非难林献堂与蔡培火等人
- 请愿运动干部长久以来一直表现出来的启蒙主义态度，面对新崛起的工农运动时，也显得左支右绌

**台湾议会请愿运动陷在以同化苦苦相逼的日本、以及高举和日本对决旗帜的民族主义派系之间，终于被撕成碎片。**

1927年，第八次请愿，依旧以悬而未决收场。

1928年，第九次请愿，请愿人数开始减少
- 这一年内地首次举行普选，因此请愿派相当期待议会的潮流变化，
- 最终不只是悬而未决收场，警戒无产势力扩张的警察态度越发专横强硬，紧张过去原本允许的散发宣传海报与小册子行动，违者直接被逮捕

1929年2月，第十次请愿，参与人数跌破2千人

1929年11月，台湾民众党召开中央常务委员会，讨论是否应该继续请愿运动
- 结论是“即使请愿一再不被接受，只要一会原因受理，运动就应该持续下去”

1930年，第十一次请愿，请愿人数只剩下1300人
- 日本政府委员表示“不管什么时候，请愿内容都是一成不变的东西，既然如此，那么历代内阁答辩的主旨也是不管何时都不会变化”

1930年，世界大恐慌、对日本共产党的极端镇压、以及自由主义衰退的形式下，在文化协会分裂后作为请愿运动基础的台湾民众党也宣告分裂。退出民众党的林献堂与蔡培火组成了和台湾当局妥协的产物——台湾地方自治联盟。

>1929年9月4日左右，美国股市暴跌，1929年10月24日股市暴跌（俗称"黑暗星期四"），大萧条成为全球新闻。<br>
1929年至1932年期间，世界国内生产总值（国内总产值）估计下降了15%。 相比之下，在2008年至2009年大萧条时期，全球GDP下降不到1%。<br>
到20世纪30年代中期，一些经济体开始复苏。然而，在许多国家，大萧条的负面影响一直持续到第二次世界大战结束。<br>
大萧条对富国和穷国都产生了毁灭性的影响。个人收入、税收、利润和价格下降，国际贸易下降超过50%。 美国的失业率上升到23%，在一些国家上升到33%。<br>
全世界的城市，特别是依赖重工业的城市，都受到了重创。许多国家的建设实际上已经停止。由于农作物价格下跌约65%，农村地区遭受了苦难。依赖第一产业（如采矿和伐木）的地区受影响最大。

1931年，第十二次请愿，依旧是驳回
- 同样想帝国议会提出、由地方自治联盟主张的地方制度改革请愿，却得到了“参考送付”的结果

1932年，第十三次请愿，人数达到了历史最高峰，超过2600名的群众参与了请愿，依旧是以失败告终。

1933年，第十四次请愿，请愿人数仅有1800余名，失败告终。

1934年，第十五次请愿，不到1200人参与，失败告终。经过协商后，蔡培火决定停止这项持续了13年之久的运动。

1936年，地方自治联盟也宣告解散。

**至此台湾人的合法政治活动名实具亡**

### 对于请愿运动，请愿方和支配方认知上的差异：
- 请愿者认为：请愿运动是在日本的主权下，为了确保立法权在内的文化和政治独立性而设置议会，并以“国民的权利”为号召进行请愿
- 支配方认为：请愿运动是彻头彻尾的民族独立运动的伪装
- 台湾内部的左派认为：请愿运动是向支配者靠拢的改良主义，并大加批判

**这个运动虽然通过置换概念，把台湾人置于既是“日本人”又非“日本人”的位置上，正因如此，它也让自己处于强迫台湾人成为“日本人”的同化论、与要求和“日本人”完全分离的民族自决主义的夹缝之间，从而受到了两方的敌视。**

**日本方面来说，平等只有身为“日本人”的情况下进行统合才能实现，主张政治上独立的人，都是要从“日本人”分离出来的民族主义者。**

## 总结
>请愿运动，本身就是一个伪命题，身为弱者，面对强权者的高压统治，要么奋起反抗——不是打倒他们，就是被他们打倒；要么默默忍受直到死亡，没有折中方案可供选择。<br>
原本想用驱虎吞狼，或是合纵连横来形容请愿者，但是到了最后决定用白日梦来形容。<br>
请愿初期，日本国内议会希望收回总督府的特权，总督府则是希望保持其特权，这个时候的请愿者通过日本议会与总督府的矛盾，将总督府定义为敌人站在了日本议会这边，并提出了“废除六三法”的行动方针，如果能够坚持，说不定这项运动还有一些希望。<br>
但是等受到了殖民政策学影响后，决定使用对自身利益更大的“自治主义”进行论述，殊不知这个论述将原本身为日本议会天然盟友的请愿者变成了日本议会、台湾总督府、请愿者三方对立的情况。而且还是当中势力最为弱小的一方。<br>
永久法《法三号》实施以后，台湾总督府保全了自身的特权利益，开始腾出手来收拾请愿运动。而随着《法三号》的实施，对于日本议会方面来说，虽然总督的立法权被削弱但并不是完全禁止，这种明显违宪的法律的实施对于意图跻身于“文明国”的日本而言是一个污点。原本的三足鼎砥局势瞬间被打破，面对联合起来的双方，毫无抵抗能力的请愿运动失败已经是不可挽回了。<br>
在13年的请愿过程中，《台湾青年》杂志的表现功不可没，可是也因为《台湾青年》杂志的出现暴露出他们对待台湾人民的态度——从杂志内容一多半都是日语——忽视又或者说是不屑一顾。如果真的是重视台湾人民，那么为什么不把日语部分的内容翻译成汉语？或者说杂志的全部内容用日语和汉语分别印刷一遍合并发行？甚至我们不能保证剩下一半非日语的部分是不是都是汉语？说不定还有英语和法语？<br>
至于在1923年到1926年间，文化协会举办了798场演讲，听众累计30万人，又有什么用呢？高高在上的他们又是否真的听到了民众的呼声？不过是想让这些民众充当他们的工具罢了。<br>
持续13年的请愿运动，参与人数最多的时候也仅仅只有2600余人，而1920年台湾接近400万的常住人口，1930年更是有460万的常住人口，到了1935年，常住人口增长到了521万。<br>
2,600:4,600,000这种悬殊的比例还好意思开口说请愿运动是“国民的要求”，这种不要脸的境地，自愧不如。
