# 第十三章 异身同体的梦想——设置台湾自治议会请愿运动

## 引言部分
- 1920年前后，台湾开展了为了反抗日本总督的压迫和获取政治权而进行的政治运动——台湾自治议会设置请愿运动，该运动被称为`过渡时期诞生的稳健型抗日运动`。
- 在1920年，日治初期，台湾以传统价值观为根基的武力抗日斗争陷入绝路，而共产主义与工人、农民运动即将兴起。
- 起始于1914年，在1920年中期到达高峰，1934年结束，该运动是当时的台湾人`唯一现代化且合法的政治活动`，目的是向日本帝国议会请愿，希望能够在台湾设立自治议会。
- 是一种以最初始的形式探索有别于和`既存的民族国家之间进行统合`或是`建立新的民族国家`之间的第三条路。

### 如何理解“过渡时期诞生的稳健型抗日运动”？
```
个人看法：
过渡时期指的是1920年前后，台湾以传统价值观为根基的武力抗日斗争陷入绝境，共产主义和工农运动方兴未艾。
稳健型指的是以和平请愿，非暴力的方式将自身的政治诉求传递出去。
抗日指的是这项请愿活动的目的是为了打破日本总督府在台湾实行的独裁统治，使得台湾人获得作为“日本人”的权利。
```

## 为获得权力的“同化”
### 台湾议会设置请愿运动的前身——台湾同化会
台湾同化会的一生：
- 1907年，梁启超和林献堂相识于日本，梁启超告诫林献堂“中国今后30年间没有救援台湾的余力，因此必须效法爱尔兰，在武力斗争败北的同时，改为向英国舆论呼吁陈情，以合法运动的方式来获得参政权。”
- 林献堂之后又从某位国民党干部了解到“台湾住民为了提升地位而向日本陈情，以获得理解和支持，这种文件的运动最为恰当。”
- 1913年春天，林献堂在东京拜访时任内务大臣原敬，失败。
- 在拜访原敬失败后，改为拜访以自由民权运动为旗手的板垣退助。板垣退助对台湾人民的遭遇深表同情。
- 1914年2月，板垣退助视察台湾之后，提议设立以“本国和台湾融合”为宗旨的台湾同化会。大隈重信和东乡平八郎等人表示支持。
- 1914年11月，林献堂拜访内田民政长官，长官叮嘱“同化会立意虽好，但政治就像利刃，如果任由小孩随意玩弄，那是极度危险的”。
- 1914年12月，板垣退助再次造访台湾，并以会长的身份在台北成立了以“内台融合”为宗旨的台湾同化会，成立之后集结了三千余名参加者。首相大隈和总督府民政长官内田嘉吉送来贺词，总督府诸多官员也出席了仪式。
- 1914年12月底，板垣退助离开台湾后，总督府以“经运不正”为由逮捕了理事。
- 1915年1月，总督府以“有害公安”之故，强行解散了同化会。

林献堂在成立同化会之初，四处游说日本中央人士，其观点内容大致为：“自日本领有台湾以来，台湾总督便凭借着‘六三法’掌握独裁权，使得台湾变成‘宛如和日本帝国毫无关系，而是总督府的台湾’。尽管天皇高唱‘一视同仁’，但总督府依旧不断镇压台湾人民，和邻近由美国统治的菲律宾相比，也仅有基础教育设施。若再放任这种状态下去，‘不管是为了台湾，还是为了日本国家，最后都不是良好之策’。”
```
在这里，林献堂把帝国议会希望收回独裁权的总督府当成了敌人，表示了身为台湾人民的代表希望和帝国同进共退拥护天皇的忠贞思想，获得了日本中央人士的支持。
```
对林献堂到访表示关心的政治家包括了尾崎行雄、岛田三郎、犬养毅、大隈重信等。和接受谢花升陈情的政治家是同一批人，都是对六三法不满的政治家，但对于林献堂和台湾人民的境遇也仅仅是口头上表示同情。

### 同化会会长板垣退助、以林献堂为代表的台湾人、台湾总督府各自的立场：
板垣退助的思想是典型的大亚洲主义（泛亚主义），将自己和大日本帝国描述成反种族歧视和反帝国主义的斗士，因为总督府的压迫导致了日台融合进展缓慢，希望安定台湾统治获得台湾民心，从而将台湾拉入日本阵营，从而使之成为日中亲善的桥梁，共同抵抗欧美白人，根本目的是为了获得对天皇忠心的臣民，毫不关心台湾人的政治权利。其政治思想为1. 强化台湾人国语和修身教育。2. 迁移至少百万内地人去台湾定居通婚。3. 缓和总督府压迫政策，使内地人和台湾人相融合。4. 日本领有台湾之目的是为了保障国家安全。

以林献堂为代表的台湾人在孤立无援的情况下，通过重新解读“一视同仁”这一个国家基本精神，获得了政治正确的诉求，即日本方面无法直接的否定台湾人民的同化请求。但是，因为其政治诉求的根本目标是获得台湾人自己的独立议会，所以这种与总督府、帝国中央议会相冲突的政治诉求从根本上而言是不可能实现的，看似如火如荼的台湾同化会为什么只能生存一个月，为什么持续了13年的台湾请愿运动最终失败之根源就在于此。

台湾总督府面对台湾人民提出的饱含政治正确的口号“日台融合”，无法直截了当地提出反对，如果胆敢明面上提出反对意见，必然会被帝国中央议会找到把柄和借口，从而顺理成章的收回总督府的独裁权、立法权，废除六三法等。所以在面对拥有国内颇具影响力的政治家支持却依旧处于绝对弱势地位的台湾人所成立的台湾同化会，总督府只需等待国内政治家回国后，便可以随心所欲地以符合规则的方式强行解散台湾同化会，从而保全自身的政治利益。


### 如何理解“台湾人居然会为了和总督府斗争，高举天皇‘一视同仁’旗帜，还向大日本帝国的中央政治家陈情，这种手法乍看之下相当不可思议”
```
之所以乍看之下觉得“相当不可思议”，是因为从整体而言，日本和台湾是对立的状态，即对台湾人民而言，日本总督、日本议会、日本政治家都是敌人。
但是正如梁启超所言“中国今后30年间没有救援台湾的余力”，台湾在二十世纪前中期，处于孤立无援四面楚歌的情况下，面对日本总督府的压迫，面对武力斗争的失败，只有在日本国内寻求和总督府敌对的政治集团的庇佑，才有能力去反抗总督府的压迫从而改善台湾人民水深火热的生活。
毕竟敌人的敌人就是朋友。
```

### 如何理解“这一运动理所当然在某种意义上在‘抗日’的同时，又带有‘亲日’的微妙性质”
```
抗日：台湾的总督府这个政府机构是日本设立的，因为反抗总督府的压迫，所以说是一种“抗日”运动。
亲日：孤立无援的台湾为了反抗总督府，只能以加快日台融合的方式，在日本国内寻求帮助，所以说是“亲日”运动。
```

## 对多样性的期盼
台湾传统教育到现代化教育的转换：
- 1898年-1918年，台湾传统教育机构书院的数量锐减20%
- 1912年-1920年，台湾公立学校就学率从6.6%提升至25.1%

台湾文化发生的转变：
- 1900年，成立天然足会
- 1911年，成立断发不改装会

台湾与琉球：
- 1900年，冲绳青年之间流行和服
- 1920年，台湾青年之间流行西装和上海服

台湾作为“文明化”的一环，应该要对老旧的风俗进行改良，对于上述转变，日本知识分子和总督府都表示了支持。但由于中国也有断发放足等运动的兴起，所以断发放足并不是有力的证据能够证明台湾“日本化”。对于日本而言，台湾青年脱掉“支那服”换上西装仅仅是“文明化”但并不等于“日本化”，至少是改穿`和服`这个阶段才行。

1915年，西来庵武装抗日运动，以日本方面大量虐杀和处死九百人的结果告终，标志着台湾以`传统文化为根基`的`武装反抗运动`走向灭亡，开始转变为以`现代化为根基`的`合法且现代`的`和平请愿运动`反抗形式，其先驱是`台湾同化会`以及`中学设置运动`。

作为被称之为台湾最现代化的阶层——内地留学生们，对中学设置运动最为敏感。因为对高等教育的追求，内地留学生群体在 1915年-1919年-1922年期间，人数从300余人增加到800余人再到超过了2400人。

在英法等国的殖民地，从宗主国留学归来的人物多半成为现代抵抗运动的核心。在日本，则出现了：
- 冲绳推动参政权获得运动的谢花升就是从内地留学归来的
- 朝鲜三一运动的独立宣言，是由朝鲜的内地留学生独自起草的
- 台湾议会设置运动，是由台湾的内地留学生推动的

在`1919年之前`，台湾留学生群体虽然都“很努力和内地风俗习惯进行同化，对于社会问题、政治问题并没有表现出深刻的关心”，即便是被人嘲笑也没有人挺身而出进行反驳，尽管大家对台湾现状感到不满却无法提出一个有效的运动仿真。

`1919年之后`，受第一次世界大战结束后民族自决的风潮，朝鲜三一运动以及中国动向的影响，又因为台湾不具备原本是独立国家的历史脉络，再加上流血镇压的过往记忆，所以台湾留学生群体没有像朝鲜人那样高唱“马上独立”，而是持续着合法与非暴力的路线。

### 如何评价台湾留学生的行为？
```
在转变前，台湾留学生群体的不作为与不敢作为是因为其生活条件的优渥导致的，无论外界如何变化，只要能够保证其优渥的生存环境和学习环境，生活在象牙塔中的学生是不会在意外界一切事物的。
大部分留学生学成返乡，开始发现他们虽然还是处于受人尊敬的位置，但是丧失了参政议政的权利。从受人尊敬的社会精英跌落人间成为了丧失主权寄人篱下的平民，这种巨大的转变加上外界民族自决民族独立运动的兴起，诱发了这种转变的发生，希望通过一种社会运动进行社会变革。
我们都知道，革命不是请客吃饭，没有不流血的革命。革命就是用暴力的方式从已有的利益集团手中抢夺和再分配利益。自认为天之骄子的台湾留学生群体，自认为高人一等，不愿意和社会底层的泥腿子一起流血牺牲，异常惜命的他们希望通过用合法与非暴力的请愿运动去祈求既得利益者施舍一部分利益，对于既得利益者而言这无异于钝刀割肉，所以是绝不可能成功的。
身为小资产阶级的台湾留学生群体其阶级的软弱性与妥协性，在这个运动里体现的淋漓尽致。
```

新民会和《台湾青年》杂志：
- `新民会建立的起因`：1918年，东京中华料理店，林献堂和留学生们一致认同“不管同化还是非同化，六三问题始终都束缚着台湾人，应该要将它撤销才对。”
- `新民会和《青年杂志》的诞生`：
    - 1920年1月，台湾留学生组成了“新民会”
    - 1920年7月，“新民会”发行了机关刊物《台湾青年》
- `新民会纲领`：仅提倡“文化向上的提升”，但在干部会议上决定开展“台湾统治的改革运动。”
- `新民会主张`：废除六三法
- `《台湾青年》杂志停刊`：因后来改名为《台湾新民报》，自动停刊
- `《台湾青年》杂志特征`：
    - 日本方面投稿者占据了不小的比例
    - 超过一半以上的内容以日文呈现
    - 台湾人的投稿也是日文居多
    - 以对话路线为主轴
- `《台湾青年》杂志的立场`：既要推进“文明化”，又要拒绝“日本化”

### 如何理解《台湾青年》杂志内容以日文居多的现象？
```
外部客观因素：该杂志受到内务省与台湾总督府的双重检阅，创办“新民会”的留学生群体是接受了日语现代教育的一群人。
内部主观因素：因为《台湾青年》杂志的发行者和投稿者是留学生、通过日语教育吸收“文明”的一群人、日本国内学者，所以在这些人的潜意识里，将“日本化”和“文明化”作了比较，虽然“文明化”不一定是“日本化”，但是“日本化”却是“文明化”的一部分。所以是用日文发表文章是文明的象征，是知识分子的象征。
后果：因为一多半的内容是用日语表达的，所以这本杂志在台湾的普及就受到了很大的限制，对于绝大多数人而言，这本杂志的内容是看不懂的。因为看不懂杂志所表达的思想，所以留学生群体的思想对于台湾人民来说是接受不了的，因为他们不懂。也就是说在留学生群体的家乡——台湾，绝大多数民众是不会支持他们的思想和行动的，我们也可以从请愿运动参与人数最多也没有超过三千人这份数据中分析出来，他们已经严重的脱离了人民群众，他们所倡议的一切行为都是空中楼阁，必将遭受失败。
大半内容用日语编写的以对话路线为主轴的《台湾青年》杂志是谁和谁对话？是留学生和日本的对话，或许在留学生眼里大部分台湾人民并不是“文明人”，不具备“文明化”的人是没有资格参与对话的。
```

### 日本人投稿强调的内容
以对话路线为主轴的《台湾青年》杂志上投稿的日本作者，类型相当广泛，反映了三一运动的氛围，几乎一致认为朝鲜和台湾的统治机制到了必须要改革的时候了。
- 植村正久和海老名弹正等基督教知识分子
    - 海老名弹正批判：“台湾教育设施不够完善，对于欲跻身世界文明国之列的日本来说是极度不像话的。”
- 永井柳太郎和岛田三郎等民权派政治家
    - 永井柳太郎主张：“日本和台湾的关系，和英国对印度的关系不同，是属于同一文化系统的同一人种。”
    - 岛田三郎表示：“日本天皇的圣意是对所有人平等的施予善政，所以日本面对西洋诸国提倡的普世人种平等的观念，应该使用于台湾。”
- 吉野作造和末弘严太郎等大正民主派
    - 因提倡“一视同仁”和“文明”的普及来化解歧视论，与同化主义的批判同时并立，从而呈现出一种`文化多元主义`的主张
- 安部矶雄和佐野学等社会主义者
- 三宅雪领与安罔正笃等亚洲主义着
- 台湾总督田健治郎以及总督府总务长官下村宏
- 早稻田大学教授内崎作三郎
    - 内崎作三郎表示：“日本民族是混合民族，应当促进日台融合。”

日本作者论述的内容，主要集中在
- “一视同仁”式的反歧视批判
- 反对六三法问题以来的总督府特权
- 高唱对抗欧美的大亚洲主义
- 主张把台湾当作“日中亲善”的媒介来重视

### 台湾人投稿强调的内容
从结论来看，台湾青年当初的论调表面上和日本方面并没有太大的差异，主要强调的是第一次世界大战后“民主”与普遍的“文明”，并对总督统治的专制性质加以批判。与日本作者的不同之处在于大多数人认为“台湾文化远远无法和世界文化相比，至少落后了2、3个世纪，保持着台湾恐怕会成为文明的落伍者的危机感。”

高举“去除陋习，吸收新文明”的旗帜，内容包括：
- 家族制度的腐败
- 卫生思想的地下
- 鸦片吸食的陋习
- 女性地位的低下
    - 《台湾青年》杂志几乎每一期都有关于提升台湾女性地位的投稿被刊登出来
    - 反对：畜妻制、买卖婚姻、缠足等制度习俗
    - 提倡：恋爱自由、婚姻自由、女子教育普及

汉民族认同的主张则是并不把日本看作“文明”的体现者。他们和那些企图给家乡带来“文明化”却又对宗主国保持着反抗心理的人有着相同的特征，他们认为：
- 日本的一切都是模仿的
    - 文化、政治、法律、文学、美术、音乐
- 保持着否定“日本化”的态度
    - 女性地位的提升比起国语（日语）的普及更为重要
    - 和服与生鱼片都不适合台湾的亚热带气候，在台湾推广和服与生鱼片是对重视卫生与机能的“文明化”大开倒车

台湾留学生群体几乎都是台湾原住民当中的上层阶级，所以他们提倡推进“文明化”，又拒绝“日本化”。
- 虽然批判同化政策，但对总督府推行的经济开发和卫生整顿政策表示支持
- 一边批判教育片中日本语，同时又感叹日语识字率低落

### 蔡培火投稿强调的内容
蔡培火初到东京对日本满怀“民族的憎恶之情”，后受日本基督教牧师植村正久的感化放下了这种感情，加入基督教以“四海兄弟主义”为信条，认为台湾是日本的一部分。

蔡培火主张在达成制度平等的同时，也要保障善良的文化。为了将“文明化”和“日本化”分离开来，他在《台湾青年》杂志发表一篇名为《吾人的同化观》的论著。他认为同化分为`自然同化`和`人为同化`，人为同化方面的内容无论怎样努力都不会成功。成功的唯一方法就是保持着“异身同体的精神”，去“尊重个性、保障善良的文化”，并抛弃过去的歧视观念和领土的野心。在他看来，现实的台湾统治却是充满了歧视的，总督府所谓的同化不过是“榨取主义的别名”、“愚民政策的表面招牌”罢了。
- 自然同化
    - 废除缠足、禁吸鸦片、等`本质性的陋习`，以及促进`人格尊重自由平等`，这些以`全世界规模开展`的事物
- 人为同化
    - 奖励和服、街名改为内地风格、门口竖立门牌

### 文明化与日本化的关系
```
文明化指的是：学习先进的文化、知识，学习科学的卫生习惯，提升女性地位，普及教育，提倡自由、平等、公正、法制等。
日本化指的是：有选择地学习先进的文化、知识、卫生习惯、衣着、普及教育等，着重学习日本传统文化、日本语、饮食习惯、衣着审美等一系列日本风味浓厚的相关内容，从而实现文化上的融合。
日本方面在面对殖民地的时候将“日本化”和“文明化”划等号，希望将当地原住民从身穿本地传统服饰转变为西装领带，再转变为和服。通过这种形式的转变，让日本是祖国的观念、忠于天皇的观念深入当地人心，实现民族的同化和融合，方便后续的统治。
台湾留学生群体几乎都是台湾原住民当中的上层阶级，他们反对将“文明化”和“日本化”划等号，希望通过“文明化”来获取知识和力量，又反对“日本化”来追求和保全自身的权利。没有人会希望自己的头上有一个掌握着自己生死大权的统治者。
```

### 留学生、总督府、日本议会各自理解的自治是什么
```
留学生的自治：指的是台湾人自己的自治
总督府的自治：指的是日本总督拥有对台湾的立法权和独裁权
日本议会的自治：指的是台湾的行政级别调整为县级，最高长官由文官担任且无条件听从内阁的安排
```
